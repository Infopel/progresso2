# Bitnami Redmine Stack 3.4.6-5

O **Bitnami Redmine Stack** é um pacote pré-configurado para simplificar a instalação e execução do Redmine, uma aplicação web de gerenciamento de projetos escrita em Ruby on Rails. O pacote inclui Apache, MySQL, Subversion, Ruby, entre outros, todos integrados e configurados.

## Principais Características

- **Fácil de instalar**: Automatiza todo o processo de instalação e configuração.
- **Independente**: Não interfere em softwares existentes no sistema.
- **Relocável**: Pode ser instalado em qualquer diretório.
- **Integrado**: Pronto para uso após a instalação.

## Componentes Principais

- **Redmine**: 3.4.6
- **Apache**: 2.4.37
- **MySQL**: 5.6.42
- **Ruby**: 2.3.3
- **Rails**: 4.2.8

## Requisitos do Sistema

- Processador x86
- 512 MB de RAM
- 400 MB de espaço em disco
- Compatível com:
  - Linux
  - Windows (32-bit)
  - macOS

## Instalação e Uso

1. Baixe o instalador no site da Bitnami: [https://bitnami.com/stacks](https://bitnami.com/stacks).
2. Escolha a pasta de instalação e configure as portas (por padrão: **Apache**: 8080, **MySQL**: 3306).
3. Após a instalação, acesse o Redmine no navegador em: `http://127.0.0.1:8080`.

## Administração

- O usuário e senha de administrador são definidos durante a instalação.
- Serviços podem ser gerenciados utilizando:
  - **Linux**: Scripts como `ctlscript.sh`.
  - **Windows**: Atalhos no menu Iniciar.
